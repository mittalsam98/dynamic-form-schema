## Getting Started

1. Clone the repo:
```bash
git clone <repository-url>
cd dynamic-form-schema
```

2. Install dependencies:
```bash
npm install
# or
yarn install
```

3. Run the development server:
```bash
npm run dev
# or
yarn dev
```

4. Open [http://localhost:5173](http://localhost:5173) in your browser to see the live application.
